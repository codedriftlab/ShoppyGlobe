# ShoppyGlobe
ShoppyGlobe is an e-commerce web application built with React. It allows users to browse products, add items to a cart, manage quantities, and place orders. It uses Redux for state management and fetches product data from an API.

# 🛒 ShoppyGlobe - E-commerce React App

##  Project Overview

ShoppyGlobe is a basic e-commerce web application built using **React + Vite**.
Users can browse products, view details, add items to cart, and place orders.

---

## Features

* Home page with product listing
* Product detail page
* Add to cart functionality
* Increase / ➖ Decrease product quantity
* Remove items from cart
* Checkout page (dummy form)
* Order placed message
* 404 Not Found page
* Fast performance with Vite

---

##  Tech Stack

* React (Vite)
* Redux Toolkit
* React Router DOM
* Tailwind CSS
* JavaScript (ES6)

## Installation & Setup

1. Clone the repository:

```
git clone <https://github.com/codedriftlab/ShoppyGlobe.git>

```

2. Navigate to project folder:

```
cd shoppyglobe
```

3. Install dependencies:

```
npm install
```

4. Run the project:

```
npm run dev
```

---

## Build for Production

```
npm run build
